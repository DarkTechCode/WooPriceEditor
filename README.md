# Редактор цен WooCommerce

Полнофункциональный WordPress плагин для работы с ценами товаров WooCommerce.

## Описание

Редактор цен WooCommerce — это мощный плагин для массового редактирования цен и других характеристик товаров WooCommerce с использованием удобного полноэкранного интерфейса.

## Структура плагина

Весь код плагина расположен в папке `woo-price-editor/`:

```
woo-price-editor/
├── assets/                      # Стили и JavaScript
│   ├── css/
│   │   ├── editor.css          # Стили редактора
│   │   └── settings.css        # Стили страницы настроек
│   └── js/
│       └── editor.js           # Frontend JavaScript
├── docs/                        # Полная документация
│   ├── installation.md         # Руководство по установке
│   ├── capabilities.md         # Права доступа
│   ├── settings.md             # Документация по настройкам
│   ├── ajax-endpoints.md       # Справочник AJAX API
│   └── technical-notes.md      # Технические заметки

│   ├── class-wpe-plugin.php    # Основной класс плагина

│   ├── class-wpe-api.php       # REST API конечные точки
```

## Быстрый старт

### Требования

- **WordPress**: 6.0 или выше
- **PHP**: 7.4 или выше
- **WooCommerce**: 3.0 или выше
- **Право доступа**: `manage_woocommerce`

### Установка

1. Загрузите папку `woo-price-editor` в `/wp-content/plugins/`
2. Активируйте плагин через меню **Плагины** в WordPress
3. Перейдите на **Редактор цен** в меню администратора
4. Настройте параметры в **Редактор цен** → **Настройки**

## Основные возможности

- ✓ Полноэкранный интерфейс без отвлекающих элементов
- ✓ Встроенное редактирование цен и характеристик товаров
- ✓ Расширенная фильтрация и поиск
- ✓ Защита на основе прав доступа
- ✓ AJAX операции с надлежащей валидацией
- ✓ Полностью локализирован на русский язык
- ✓ Полная документация и набор тестов
